## Sequential Logic

### Latches and Flip-Flops

#### 触发器模板

异步复位：只要 reset 变化，寄存器立刻复位，不需要等时钟。

| 特性       | 同步复位         | 异步复位                          |
| ---------- | ---------------- | --------------------------------- |
| 是否等时钟 | 需要             | 不需要                            |
| 敏感列表   | `@(posedge clk)` | `@(posedge clk or posedge reset)` |
| 复位时机   | clk边沿          | reset变化立即                     |
| 电路实现   | 逻辑控制         | 直接清零触发器                    |

| 类型              | 敏感列表                       | 条件         |
| ----------------- | ------------------------------ | ------------ |
| active high async | `posedge clk or posedge reset` | `if(reset)`  |
| active low async  | `posedge clk or negedge reset` | `if(!reset)` |
| active high sync  | `posedge clk`                  | `if(reset)`  |
| active low sync   | `posedge clk`                  | `if(!reset)` |

- 异步 reset → 写在敏感列表（异步复位要不等时钟立即执行。）
- 同步 reset → 不写

#### 怎么从“软件思维”切回“电路思维”

你以后做时序逻辑题，可以固定按这三步想：

##### 第一步：先问自己，寄存器下一拍要变成什么？

也就是：

```
Q(next) = ?
```

##### 第二步：如果是 DFF，就直接令

```
D = Q(next)
```

##### 第三步：把 `Q(next)` 化成组合逻辑表达式

然后写成：

```verilog
assign d = ...
always @(posedge clk)
    Q <= d;
```

**组合逻辑 + 状态寄存器 + 反馈**

| 触发器类型   | 关系                                 |
| ------------ | ------------------------------------ |
| D flip-flop  | `D = Q(next)`                        |
| T flip-flop  | `T = Q ⊕ Q(next)`                    |
| JK flip-flop | `J = ~Q · Q(next)``K = Q · ~Q(next)` |
| SR flip-flop | `S = ~Q · Q(next)``R = Q · ~Q(next)` |

#### 从“瞬时检测”到“状态保持” (Capture Logic)

##### 1. 核心逻辑公式

在 Verilog 中，如果你想让一个信号**一旦触发就锁死（除非复位）**，最经典的表达式是：
 $out \Leftarrow (out \text{ | } \text{Trigger\_Signal})$

- **`out`（左边）**：下一拍的状态。
- **`out`（右边）**：当前的记忆（如果已经是 1，则保持 1）。
- **`Trigger_Signal`**：新发生的事件（如果是 1，则把记忆刷成 1）。

##### 2. 复位优先（Precedence）

题目要求“如果复位和触发同时发生，复位优先”。这在硬件中通过 `if-else` 的层级直接实现：

```verilog
always @(posedge clk) begin
    if (reset) 
        out <= 32'b0;         // 优先级最高：强制清零
    else 
        out <= out | (prev & ~in); // 维持现状 或 捕获新边沿
end
```

**“下一拍的值，等于‘现在的状态’和‘新发生的跳变’取并集。”**
 只要 `out` 曾经变成过 `1`，那么 `1 | (任何东西)` 永远等于 `1`。这就是硬件实现**“锁存”**或**“记忆”**最基本的方法。

##### 3. 两种截然不同的电路

| 代码写法                     | 硬件模型       | 行为描述                                                     |
| ---------------------------- | -------------- | ------------------------------------------------------------ |
| `out <= (prev & ~in);`       | **边沿检测器** | 信号变动后的那一拍，`out` 闪一下 `1`，随后立即变回 `0`。     |
| `out <= out | (prev & ~in);` | **状态捕获器** | 一旦检测到变动，`out` 变成 `1` 并**保持住**，直到 `reset` 信号强制把它冲刷成 `0`。 |

#### 双边沿检测的“异或方案”

| 步骤                   | 逻辑操作            | 物理意义                                                     |
| ---------------------- | ------------------- | ------------------------------------------------------------ |
| **第一步：上升沿分支** | `p <= d ^ n;`       | 在上升沿，我们把输入 $D$ 和下降沿的结果做异或，存入寄存器 $P$。 |
| **第二步：下降沿分支** | `n <= d ^ p;`       | 在下降沿，我们把输入 $D$ 和上升沿的结果做异或，存入寄存器 $N$。 |
| **第三步：最终输出**   | `assign q = p ^ n;` | 最终的 $Q$ 是这两个寄存器的异或值。                          |

**为什么这个魔法能成功？**

- 当**上升沿**到来时，$P$ 更新为 $d \oplus n$。此时输出 $q = p \oplus n = (d \oplus n) \oplus n$。根据异或定律，结果恰好就是 $d$！
- 当**下降沿**到来时，$N$ 更新为 $d \oplus p$。此时输出 $q = p \oplus n = p \oplus (d \oplus p)$。结果同样恰好是 $d$！



### Counters

时钟程序

> Create a set of counters suitable for use as a 12-hour clock (with am/pm indicator). Your counters are clocked by a fast-running `clk`, with a pulse on `ena` whenever your clock should increment (i.e., once per second).
>
> `reset` resets the clock to 12:00 AM. `pm` is 0 for AM and 1 for PM. `hh`, `mm`, and `ss` are two **BCD** (Binary-Coded Decimal) digits each for hours (01-12), minutes (00-59), and seconds (00-59). Reset has higher priority than enable, and can occur even when not enabled.

```verilog
module top_module(
    input clk,
    input reset,
    input ena, 
    output reg pm,
    output reg [7:0] hh, 
    output reg [7:0] mm,
    output reg [7:0] ss
);

    always @(posedge clk) begin
        if (reset) begin
            pm <= 1'b0;
            hh <= 8'h12;
            mm <= 8'h00;
            ss <= 8'h00;
        end
        else if (ena) begin
            if (ss == 8'h59) begin
                ss <= 8'h00;

                if (mm == 8'h59) begin
                    mm <= 8'h00;

                    if (hh == 8'h11) begin
                        hh <= 8'h12;
                        pm <= ~pm;
                    end
                    else if (hh == 8'h12) begin
                        hh <= 8'h01;
                    end
                    else if (hh[3:0] == 4'h9) begin
                        hh[3:0] <= 4'h0;
                        hh[7:4] <= hh[7:4] + 4'h1;
                    end
                    else begin
                        hh[3:0] <= hh[3:0] + 4'h1;
                    end
                end
                else begin
                    if (mm[3:0] == 4'h9) begin
                        mm[3:0] <= 4'h0;
                        mm[7:4] <= mm[7:4] + 4'h1;
                    end
                    else begin
                        mm[3:0] <= mm[3:0] + 4'h1;
                    end
                end
            end
            else begin
                if (ss[3:0] == 4'h9) begin
                    ss[3:0] <= 4'h0;
                    ss[7:4] <= ss[7:4] + 4'h1;
                end
                else begin
                    ss[3:0] <= ss[3:0] + 4'h1;
                end
            end
        end
    end

endmodule
```



### Shift Registers

#### 循环移位

| 概念                        | 逻辑描述           | Verilog 语法示例 (8位向量 A)             |
| --------------------------- | ------------------ | ---------------------------------------- |
| **逻辑左移 (Shift Left)**   | 补 0，丢弃最高位   | `A <= {A[6:0], 1'b0};` 或 `A <= A << 1;` |
| **循环左移 (Rotate Left)**  | 最高位绕回到最低位 | **`A <= {A[6:0], A[7]};`**               |
| **循环右移 (Rotate Right)** | 最低位绕回到最高位 | **`A <= {A[0], A[7:1]};`**               |

#### 算术移位

| 特性         | 逻辑移位 (Logical Shift)            | 算术移位 (Arithmetic Shift)                        |
| ------------ | ----------------------------------- | -------------------------------------------------- |
| **符号感应** | **无**。把数据看作纯粹的 0/1 序列。 | **有**。承认最高位是“符号位（Sign Bit）”。         |
| **右移补位** | 永远补 **`0`**。                    | **复制符号位**。原最高位是 1 就补 1，是 0 就补 0。 |
| **左移补位** | 补 **`0`**。                        | 补 **`0`**（与逻辑左移基本一致）。                 |
| **数学意义** | 仅移动位，用于处理无符号数。        | 保持符号，实现带符号数的 $2^n$ 倍增或减半。        |



### More Circuits

#### 注意 HDLBits 向量方向

HDLBits 的向量通常是：

```verilog
q[511] ... q[2] q[1] q[0]
```

所以：

```verilog
左邻居  = q[i+1]
右邻居  = q[i-1]
自己    = q[i]
```
